Kanji Training

Приложение для изучения японских иероглифов (кандзи). Содержит базу данных иероглифов, карточки для запоминания, тесты по уровням JLPT и возможность создавать собственный список.

Функциональность
Просмотр карточек с иероглифами
Тесты, разбитые по уровням JLPT (N5, N4, N3)
Создание собственного списка иероглифов
Очистка пользовательского списка

Установка

Скачайте файл debug_app.apk из репозитория, откройте его на устройстве Android и разрешите установку из неизвестных источников.

Технологии
Язык: Java
Среда разработки: Android Studio
